Designing a Perfume and Cosmetics Website

  • Making my homepage

The website name is www.Mizspa.com. The purpose of this website is to allow female customers (mostly from ages 20-40) to be able to easily browse and purchase cosmetics.

Audience

The audience that Mizspa.com targets is mainly women from ages 20 to 40 from Australia or Hong Kong.

Objectives

Our objectives of this web design is to be able to contruct a website that is easy to navigate. We want the website to be able to attract more customers and other business potentials. Since the owners of the company want a website that doesn't give a "local feel" to it, we are going to build a new website from scratch that looks professional and legit.

Success Criteria

  • Website is easy to navigate and use.
  • Website design is more professional than the previous website
  • Website looks trustable and attracts customers

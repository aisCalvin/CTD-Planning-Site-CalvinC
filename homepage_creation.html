Using Fireworks to Create a Homepage

Working with Fireworks Pros and Cons
    Pros
  • Fireworks' slice tool is simple to use when making my story board for my homepage
  • Fireworks and all its tools are easy to learn and is useful for making my homepage. For example, I can use easily use arrows and text boxes to label each part of the home page.
    Cons
  • Despite the tools of Fireworks being simple and easy to use, the tools are very limited in making a more professional homepage. For example, it takes longer to create difficult designs.
  • Fireworks only allows someone to create a design, not anything else like coding.
Using Dreamweaver to Create a Homepage
Working with Dreamweaver Pros and Cons
    Pros
  • Dreamweaver has the ability to produce accurate and detailed designs that Fireworks can't. For example, in Dreamweaver, there is coding. This means that the user can use HTML and CSS to design the product they want exactly.
  • Dreamweaver's coding has different highlights for different codes, so it is easier to tell which code is which.
  • You can split your screen in Dreamweaver. This means that you can see your coding and your design at the same time. This is really usefull because you can see what your code does to your design quickly.
      • Cons
  • A con for Dreamweaver is how it looks. For example, if you are new to Dreamweaver, it looks really confusing.
